Getting Started

What is Python?

Python is one of the most popular and beginner-friendly programming languages in the world today.

  • Definition: Python is a high-level, interpreted programming language known for its clean, readable syntax.
  • Created by: Guido van Rossum, first released in 1991.
  • Fun origin: It's named after the British comedy show "Monty Python's Flying Circus" — not the snake!

Python's logo shows two snakes, but its name comes from a comedy show.

Motivation

Why Learn Python?

Python is used almost everywhere in the tech industry today.

  • Beginner-friendly: Its syntax is simple and reads almost like English.
  • Web development: Used to build websites and web applications.
  • Data science: The top choice for analyzing data and building charts.
  • Artificial intelligence: Powers many popular AI and machine learning tools.
  • Automation: Great for writing small scripts that do repetitive tasks automatically.

Getting Ready

Setting Up Your Python Environment

Before writing code, you need somewhere to write and run it.

  • python.org: The official website where you can download and install Python for free.
  • IDLE: A simple editor that comes bundled with Python — great for beginners.
  • VS Code: A popular, free code editor with helpful Python extensions.
  • Thonny: An editor built specifically for beginners learning Python.
  • Online interpreters: Websites that let you run Python code in your browser with no installation.
TipAs a beginner, an online interpreter is a great way to practice without installing anything.
Milestone

Your First Python Program

Every programmer's journey starts with the same simple line of code.

hello_world.py 
print("Hello, World!")
  • print() is a built-in function that displays text on the screen.
  • The text inside the quotation marks ("Hello, World!") is called a string.
  • Save the file with a .py extension, then run it to see the output.
Rules to Remember

Python Syntax Basics

Python has a few simple but strict rules that keep code neat and readable.

  • Indentation matters: Python uses spaces at the start of a line to group code together — unlike many other languages.
  • Comments: Anything after a # is ignored by Python and is meant for humans to read.
  • Case-sensitive: Name and name are treated as two different variables.
syntax_demo.py 
# This is a comment — Python ignores this line
print("Indentation matters!")
How It Works

Running Python Code

There are two common ways to run Python code.

  • Script mode: Write code in a .py file and run the whole file at once.
  • Interactive mode (REPL): Type one line at a time and see the result immediately — great for quick testing.

REPL stands for Read-Eval-Print Loop.

Applied Example

Example: A Tiny Python Script

Let's look at a slightly bigger example that greets the user and does simple math.

greet_and_add.py 
print("Welcome to Programming 1!")
print("2 + 2 =", 2 + 2)
OutputRunning this script prints two lines: a welcome message, and the result of adding 2 + 2.
