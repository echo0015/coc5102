Data Types and Variables

Topic 04 · How Python Stores Information

Learning Objectives

  • Define what a variable is and how to create one in Python.
  • Identify Python's core data types: int, float, str, and bool.
  • Explain the rules for naming variables correctly.
  • Convert values between different data types.
Getting Started

What is a Variable?

A variable is simply a labeled box that holds a piece of information for later use.

  • Definition: A variable is a name that refers to a value stored in the computer's memory.
  • Analogy: Think of a variable as a labeled storage box — the label is the name, and the value inside is the data.
  • Reusable: Once stored, a variable's value can be used again and again, or changed later.

A variable is like a labeled box holding a value.

Rules to Remember

Variable Naming Rules

Python has strict rules for what makes a valid variable name.

RuleValid ExampleInvalid Example
Must start with a letter or underscoreage, _score1age
Can contain letters, numbers, underscoresstudent_1student-1
Case-sensitiveName ≠ name—
Cannot be a reserved keywordtotalclass, for
TipUse clear, descriptive names like student_name instead of vague ones like x or data1.
Syntax

Assigning Values to Variables

The equals sign (=) is used to store a value inside a variable — this is called assignment.

variables.py 
age = 18
name = "Maria"
is_student = True

print(name, "is", age, "years old.")
NoteIn Python, = means "assign," not "equals" like in math. We'll cover the true equality check (==) in a later topic.

A data type tells Python what kind of value a variable is holding, and what you can do with it.

Data Type

Numeric Types: int and float

Python has two main types for storing numbers.

TypeStoresExample
intWhole numbers (no decimal point)age = 18
floatNumbers with a decimal pointprice = 19.99
numbers.py 
quantity = 5        # int
price = 12.50       # float
total = quantity * price
print(total)
Data Type

String Type (str)

A string is any text, surrounded by quotation marks.

  • Strings can use single quotes 'like this' or double quotes "like this".
  • Concatenation: The + operator joins two strings together.
strings.py 
first_name = "Juan"
last_name = "Dela Cruz"
full_name = first_name + " " + last_name
print(full_name)   # Juan Dela Cruz
Data Type

Boolean Type (bool)

A boolean can only ever be one of two values: True or False.

  • Used to represent yes/no, on/off, or true/false situations.
  • Very important for decision-making in code, which we'll explore in Control Flow.
booleans.py 
is_raining = True
is_weekend = False
print(is_raining)

Booleans are like a light switch: only ON or OFF.

Tool

Checking a Variable's Data Type

The type() function tells you exactly what data type a value is.

check_type.py 
age = 18
print(type(age))       # <class 'int'>

price = 19.99
print(type(price))     # <class 'float'>

name = "Maria"
print(type(name))      # <class 'str'>
TipUse type() whenever you're unsure what kind of value a variable holds — it's a great debugging habit.
Concept

Type Conversion (Casting)

Sometimes you need to convert a value from one data type to another.

FunctionConverts ToExample
int()Whole numberint("5") → 5
float()Decimal numberfloat("5.5") → 5.5
str()Textstr(5) → "5"
ExampleInput from users always arrives as text (str) — you often need int() to convert it before doing math with it.
Applied Example

Example: A Simple Receipt Program

Let's combine several data types in one small program.

receipt.py 
item_name = "Notebook"      # str
price = 45.50                # float
quantity = 3                 # int
on_sale = True                # bool

total = price * quantity
print(item_name, "x", quantity, "=", total)
NoteNotice how each variable stores a different data type, yet they all work together in the same program.

Did You Know?

Fun Facts About Data Types

A few fun tidbits to impress your friends.

No Size Limit

Unlike many languages, Python's int can grow as large as your computer's memory allows.

Booleans Are Numbers

In Python, True actually equals 1 and False equals 0 behind the scenes.

Dynamic Typing

Python figures out a variable's type automatically — you never have to declare it yourself.

Unicode Strings

Python strings can store emojis and characters from almost every language in the world.
