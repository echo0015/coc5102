Getting Started

What is an Operator?

An operator is a special symbol that performs an action on one or more values, called operands.

  • Definition: Operators combine values into an expression that produces a result.
  • Simple explanation: Just like in math class, + adds and - subtracts — Python uses many of the same symbols.
ExampleIn 5 + 3, the + is the operator, and 5 and 3 are the operands.

Operators combine values to produce a new result.

Concept

Operator Precedence

When an expression has multiple operators, Python follows a strict order — very similar to math class's PEMDAS.

  • Parentheses () are always calculated first.
  • Exponents ** come next.
  • Multiplication, Division, Floor Division, Modulus (* / // %) come next.
  • Addition and Subtraction (+ -) happen last.
precedence.py 
result = 2 + 3 * 4     # multiplication happens first
print(result)          # 14, not 20
Applied Example

Combining Operators in Expressions

Real programs often mix arithmetic, comparison, and logical operators together.

exam_check.py 
score = 78
attendance = 95

passed = score >= 75 and attendance >= 80
print(passed)   # True
NotePython evaluates score >= 75 and attendance >= 80 first, then combines both results using and.
Watch Out

Common Mistake: = vs. ==

One of the most common beginner mistakes is confusing assignment with comparison.

SymbolPurposeExample
=Assigns (stores) a value into a variablescore = 90
==Compares two values for equalityscore == 90
ImportantWriting if score = 90: instead of if score == 90: will cause an error in Python — always double-check your equals signs!

Did You Know?

Fun Facts About Operators

A few fun tidbits to impress your friends.

Chained Comparisons

Python lets you write 1 < x < 10 directly — most other languages require two separate comparisons.

Modulus Trick

The % operator is the easiest way to check if a number is even: num % 2 == 0.

Plus Joins Strings Too

The same + symbol adds numbers and joins (concatenates) strings, depending on the data type.

Shared Symbols

Most of Python's operators (+ - * / == etc.) are shared with many other programming languages.
