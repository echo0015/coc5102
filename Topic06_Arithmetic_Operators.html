Getting Started

What is an Operator?

An operator is a special symbol that performs an action on one or more values, called operands.

  • Definition: Operators combine values into an expression that produces a result.
  • Simple explanation: Just like in math class, + adds and - subtracts — Python uses many of the same symbols.
ExampleIn 5 + 3, the + is the operator, and 5 and 3 are the operands.

Operators combine values to produce a new result.

Closer Look

Division: / vs. //

Python has two division operators, and beginners often mix them up.

division.py 
print(7 / 2)     # 3.5   (regular division, keeps decimals)
print(7 // 2)    # 3     (floor division, drops decimals)
TipUse // when you only care about whole groups, like splitting 7 candies among 2 people evenly (3 each, 1 left over).
Concept

Order of Operations

When an expression has multiple operators, Python follows a strict order — very similar to math class's PEMDAS.

  • Parentheses () are always calculated first.
  • Exponents ** come next.
  • Multiplication, Division, Floor Division, Modulus (* / // %) come next.
  • Addition and Subtraction (+ -) happen last.
precedence.py 
result = 2 + 3 * 4     # multiplication happens first
print(result)          # 14, not 20
Watch Out

Common Mistake: Forgetting Decimal Results

Regular division (/) in Python always returns a float, even when the numbers divide evenly.

division_type.py 
result = 10 / 2
print(result)         # 5.0, not 5
print(type(result))   # <class 'float'>
ImportantIf you need a whole number result, use // or wrap the result in int().

Here's everything we covered in this topic, in one glance.

Did You Know?

Fun Facts About Arithmetic Operators

A few fun tidbits to impress your friends.

Modulus Trick

The % operator is the easiest way to check if a number is even: num % 2 == 0.

Plus Joins Strings Too

The same + symbol adds numbers and joins (concatenates) strings, depending on the data type.

No Overflow

Unlike many languages, Python's numbers can grow arbitrarily large without "overflowing."

Shared Symbols

Most of Python's arithmetic symbols (+ - * /) are shared with almost every other programming language.
