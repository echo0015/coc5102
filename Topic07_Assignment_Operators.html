Assignment Operators

Topic 07 · Storing and Updating Values

Learning Objectives

  • Use the = operator to store a value in a variable.
  • Use compound assignment operators like += to update a variable.
  • Explain why compound assignment is a useful shortcut.
  • Avoid confusing assignment (=) with comparison (==).
Getting Started

What is Assignment?

Assignment means storing a value inside a variable, using the = symbol.

  • Definition: The = operator takes the value on its right and stores it in the variable on its left.
  • Not math equals: In Python, = means "assign," not "is equal to."

x = 5 means "put 5 inside the box named x."

Reference

Assignment Operators

Compound assignment operators combine an operation with assignment, in one shorter step.

OperatorExampleSame As
=x = 5Store 5 in x
+=x += 3x = x + 3
-=x -= 3x = x - 3
*=x *= 3x = x * 3
/=x /= 3x = x / 3
//=x //= 3x = x // 3
%=x %= 3x = x % 3
**=x **= 3x = x ** 3
Why It Matters

Why Use Compound Assignment?

It's shorter to type, and it makes your intent ("update this variable") obvious at a glance.

compare.py 
# Without compound assignment
score = score + 10

# With compound assignment
score += 10
TipBoth lines do exactly the same thing — += is just a shortcut that saves typing.
Applied Example

Example: Keeping a Running Total

Compound assignment is perfect for accumulating a value over several steps.

running_total.py 
total = 0
total += 50   # bought an item for 50
total += 30   # bought another for 30
total -= 10   # got a 10 discount

print(total)  # 70
Watch Out

Common Mistake: = vs. ==

One of the most common beginner mistakes is confusing assignment with comparison.

SymbolPurposeExample
=Assigns (stores) a value into a variablescore = 90
==Compares two values for equalityscore == 90
ImportantWriting if score = 90: instead of if score == 90: will cause an error in Python — always double-check your equals signs!

Did You Know?

Fun Facts About Assignment

A few fun tidbits to impress your friends.

Multiple Assignment

Python lets you assign several variables in one line: a, b = 1, 2.

Instant Swap

You can swap two variables' values in one line: a, b = b, a — no temporary variable needed.

Right to Left

Assignment always happens right to left — the right side is calculated fully before being stored.

Shared Convention

Compound assignment operators like += exist in almost every modern programming language.
