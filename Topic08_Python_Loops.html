Python Loops

Topic 08 · Repeating Actions Without Repeating Code

Learning Objectives

  • Explain why loops are used to avoid repetitive code.
  • Write a for loop to iterate over a range or sequence.
  • Write a while loop with a clear stopping condition.
  • Use break and continue to control loop execution.
Getting Started

What is a Loop?

A loop repeats a block of code multiple times, without you having to write it over and over.

  • Definition: A loop is a control structure that repeats a set of instructions while a condition holds.
  • Simple explanation: Instead of writing print("Hi") five times, a loop can do it for you in just two lines.

Loops turn "do this once" into "do this many times."

Why It Matters

Why Use Loops?

Loops make code shorter, cleaner, and much easier to maintain.

without_loop.py 
print("Hi")
print("Hi")
print("Hi")
print("Hi")
print("Hi")
with_loop.py 
for i in range(5):
    print("Hi")
Syntax

The for Loop

A for loop repeats code once for every item in a sequence, such as a range of numbers.

for_loop.py 
for i in range(5):
    print(i)

# Output: 0 1 2 3 4
  • i is the loop variable — it changes on every repeat.
  • range(5) produces the numbers 0, 1, 2, 3, 4 — five numbers total.
Tool

The range() Function

range() generates a sequence of numbers, and can be used in a few different ways.

UsageProduces
range(5)0, 1, 2, 3, 4
range(2, 6)2, 3, 4, 5
range(0, 10, 2)0, 2, 4, 6, 8
Noterange() always stops before the last number you give it — range(5) never includes 5.
Syntax

The while Loop

A while loop repeats code as long as its condition stays True.

while_loop.py 
count = 1

while count <= 5:
    print(count)
    count += 1
ImportantThe line count += 1 is what eventually makes the condition False — without it, the loop would run forever!
Comparison

for Loop vs. while Loop

Choosing the right loop depends on whether you know how many times to repeat.

Loop TypeBest When...Example
forYou know exactly how many times to repeatPrint numbers 1 to 10
whileYou repeat until some condition changesKeep asking until user enters "quit"
Loop Control

Loop Control: break

The break keyword immediately stops the loop entirely, no matter what.

break_demo.py 
for i in range(10):
    if i == 3:
        break
    print(i)

# Output: 0 1 2
ExampleAs soon as i equals 3, the loop exits completely — the numbers 3 through 9 never get printed.
Loop Control

Loop Control: continue

The continue keyword skips the rest of the current repeat and jumps to the next one.

continue_demo.py 
for i in range(5):
    if i == 2:
        continue
    print(i)

# Output: 0 1 3 4
NoteUnlike break, the loop keeps going — it only skips printing when i equals 2.
Concept

Nested Loops

A loop can be placed inside another loop — useful for grids, tables, and patterns.

nested_loop.py 
for row in range(3):
    for col in range(3):
        print("*", end=" ")
    print()
TipFor every single repeat of the outer loop, the entire inner loop runs from start to finish.
Watch Out

Common Pitfall: Infinite Loops

If a while loop's condition never becomes False, the loop will run forever.

infinite_loop.py 
# Danger! This never stops:
count = 1
while count <= 5:
    print(count)
    # forgot count += 1
ImportantAlways double-check that something inside your while loop eventually makes the condition False.
Applied Example

Example: Summing Numbers 1 to 10

A classic beginner problem — perfect for practicing loops.

sum_numbers.py 
total = 0

for number in range(1, 11):
    total += number

print("Sum:", total)   # Sum: 55
Notetotal starts at 0 and grows by each number from 1 to 10, one loop repeat at a time.
Recap

Summary — Key Points

Here's everything we covered in this topic, in one glance.

for loops repeat a known number of times, often using range().
while loops repeat as long as a condition stays True.
break exits a loop early; continue skips to the next repeat.
Always make sure a while loop's condition can eventually become False.
