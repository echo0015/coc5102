Getting Started

What Are Logical Operators?

Logical operators combine multiple True/False conditions into a single True/False result.

  • Definition: Instead of checking one condition, logical operators let you check several at once.
  • Simple explanation: "Is it the weekend and am I free?" is two conditions joined by and.

Logical operators let conditions team up.

Reference

The Logical Operators

Python has exactly three logical operators.

OperatorMeaningExampleResult
andTrue only if both sides are True5 > 3 and 2 > 1True
orTrue if at least one side is True5 > 3 or 2 > 10True
notFlips True to False, or False to Truenot (5 > 3)False

Applied Example

Combining Comparisons with Logical Operators

Real conditions almost always combine comparison operators with logical operators.

ride_check.py 
age = 13
height = 155

can_ride = age >= 12 and height >= 150
print(can_ride)   # True
Example"You can enter the ride if age >= 12 and height >= 150" — both conditions must be true.
Big Picture

Operator Precedence: The Full Picture

Now that you know arithmetic, assignment, comparison, and logical operators, here's the order Python evaluates them all.

  • Parentheses () first, always.
  • Arithmetic (** * / // % + -) next.
  • Comparisons (== != > < >= <=) next.
  • Logical operators — not, then and, then or — evaluated last.
NoteWhen in doubt, add parentheses — they make your intent clear and never change a correct result.

Did You Know?

Fun Facts About Logical Operators

A few fun tidbits to impress your friends.

Short-Circuit Evaluation

Python stops checking as soon as it knows the answer — in or, if the first part is True, it never checks the second.

Foundation of Computing

and, or, and not come from Boolean algebra, named after mathematician George Boole in the 1800s.

Same Logic, Real Circuits

Computer chips are built from tiny physical AND, OR, and NOT logic gates — the same logic you just learned!

Everywhere in Code

Search filters, login checks, and game rules all rely on combining conditions with logical operators.
