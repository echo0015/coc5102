Topic 11 · Teaching Programs to Make Decisions

Getting Started

What is Control Flow?

By default, Python runs code from top to bottom, one line at a time. Control flow lets us change that order.

  • Default behavior: Without any decisions, Python executes every line in sequence, once each.
  • Control flow: Using conditions, we can make the program skip, choose, or repeat certain lines.

Conditions act like forks in the road for your program.

Syntax

The if Statement

An if statement runs a block of code only when its condition is True.

basic_if.py 
age = 20

if age >= 18:
    print("You are an adult.")
  • The condition age >= 18 must end with a colon :.
  • The code that runs if the condition is True must be indented.
Syntax

The if-else Statement

Use else to run a different block of code when the condition is False.

if_else.py 
age = 15

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
ExampleExactly one of the two blocks will run — never both, and never neither.
Syntax

The if-elif-else Chain

Use elif ("else if") when you need to check more than two possible outcomes.

grading.py 
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")
NotePython checks each condition from top to bottom and stops at the first one that is True.
Concept

Nested Conditions

An if statement can be placed inside another if statement — this is called nesting.

nested_if.py 
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed.")
    else:
        print("ID required.")
else:
    print("Too young to enter.")
TipNotice the inner if-else is indented one extra level — indentation shows Python exactly which block belongs to which condition.
Concept

Combining Conditions with Logical Operators

Instead of nesting, you can often combine conditions using and, or, and not.

combined_condition.py 
age = 20
has_id = True

if age >= 18 and has_id:
    print("Entry allowed.")
else:
    print("Entry denied.")
NoteThis does exactly the same thing as the nested version on the previous slide, just in fewer lines.
Applied Example

Example: A Simple Grading System

Let's trace exactly what happens when a student's score is checked.

  • Input: The student's score is 72.
  • Python checks score >= 90 → False, skip.
  • Python checks score >= 80 → False, skip.
  • Python checks score >= 70 → True! Runs print("Grade: C").
  • The rest of the chain (else) is skipped entirely.
OutputGrade: C
Watch Out

Common Mistakes with Conditions

These small errors trip up almost every beginner at some point.

  • Wrong indentation: Python uses indentation to know which lines belong to the if block — inconsistent spacing causes errors.
  • Missing colon: Every if, elif, and else line must end with a :.
  • Using = instead of ==: Comparing values requires the double equals sign.
wrong_vs_right.py 
# Wrong
if age >= 18
    print("Adult")

# Correct
if age >= 18:
    print("Adult")
Recap

Here's everything we covered in this topic, in one glance.

if runs code only when its condition is True.
if-else picks between exactly two outcomes.
elif checks additional conditions in order.
Conditions can be nested or combined with and/or/not.

Did You Know?

Fun Facts About Control Flow

A few fun tidbits to impress your friends.

Indentation Is the Law

Python is one of the few major languages where indentation isn't just style — it's required syntax.

No switch Statement

Unlike some languages, older Python versions have no switch statement — if-elif chains do that job instead.

Ternary Shortcut

Python supports a one-line if-else called a "ternary expression," e.g. "Adult" if age >= 18 else "Minor".

Everywhere in Games

Every decision a video game makes — win, lose, level up — relies on control flow just like this.
