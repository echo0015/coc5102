Getting Started

What is a List?

A list is an ordered collection that can hold many values in a single variable.

  • Definition: A list stores multiple values together, in order, under one name.
  • Simple explanation: Instead of five separate variables for five grades, one list can hold all five.
  • Mutable: Unlike strings, a list's contents CAN be changed after it's created.

A list is like a row of labeled slots, each holding one value.

Syntax

Creating a List

Square brackets and commas are all it takes to build a list.

create_list.py 
fruits = ["apple", "banana", "cherry"]
grades = [88, 92, 79, 95]
mixed = ["Maria", 18, True]

print(fruits)
NoteA single list can even hold different data types mixed together, like the mixed example above.

Accessing List Elements

Lists use the same zero-based indexing you already know from strings.

access_list.py 
fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple
print(fruits[-1])   # cherry (last item)
Syntax

List Slicing

Just like strings, lists can be sliced to grab a range of items.

slice_list.py 
numbers = [10, 20, 30, 40, 50]
print(numbers[1:3])   # [20, 30]
print(numbers[:2])    # [10, 20]
print(numbers[::-1])  # [50, 40, 30, 20, 10]
TipEverything you learned about string slicing in Topic 05 applies to lists too.
Concept

Modifying List Elements

Unlike strings, lists are mutable — you can change an item after the list is created.

modify_list.py 
fruits = ["apple", "banana", "cherry"]
fruits[1] = "blueberry"
print(fruits)   # ['apple', 'blueberry', 'cherry']
ImportantThis is the key difference from strings — fruits[1] = "blueberry" works fine, but the same line on a string would cause an error.
Syntax

Adding Elements

Lists can grow after they're created, using append() or insert().

MethodWhat it Does
append(x)Adds x to the end of the list
insert(i, x)Adds x at position i, shifting the rest over
add_to_list.py 
fruits = ["apple", "banana"]
fruits.append("cherry")
fruits.insert(1, "kiwi")
print(fruits)   # ['apple', 'kiwi', 'banana', 'cherry']
Syntax

Removing Elements

Python gives you a few different ways to remove items from a list.

MethodWhat it Does
remove(x)Removes the first item that matches value x
pop(i)Removes and returns the item at position i (last item if i is left blank)
del list[i]Deletes the item at position i
remove_from_list.py 
fruits = ["apple", "banana", "cherry"]
fruits.remove("banana")
print(fruits)   # ['apple', 'cherry']
Reference

Common List Methods

A handful of built-in methods handle almost everything you'll need to do with a list.

Method / FunctionWhat it Does
len(list)Counts how many items are in the list
sort()Sorts the list in place, smallest to largest
reverse()Reverses the order of the list in place
count(x)Counts how many times x appears
index(x)Finds the position of the first x
Concept

Looping Through a List

A for loop can visit every item in a list, one at a time.

loop_list.py 
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
TipThis is by far the most common way lists and loops (Topic 09... now Topic 12) are used together in real programs.
Syntax

Checking Membership with in

The in keyword checks whether a value exists anywhere in a list.

membership.py 
fruits = ["apple", "banana", "cherry"]

print("banana" in fruits)      # True
print("mango" in fruits)       # False
Exampleif "mango" in fruits: combines membership checking directly with the conditions you learned in Topic 08.
Concept

Nested Lists

A list can even contain other lists — handy for grids or tables of data.

nested_list.py 
grid = [
    [1, 2, 3],
    [4, 5, 6]
]

print(grid[0])      # [1, 2, 3]
print(grid[0][1])   # 2
Notegrid[0][1] means "get row 0, then item 1 from that row" — two indexes, one for each level.
Applied Example

Example: Averaging a List of Grades

Combining a list with a loop is one of the most useful patterns in programming.

average_grades.py 
grades = [88, 92, 79, 95]
total = 0

for grade in grades:
    total += grade

average = total / len(grades)
print("Average:", average)   # Average: 88.5
Recap

Summary — Key Points

Here's everything we covered in this topic, in one glance.

Lists store multiple values, in order, under one variable name.
Lists are mutable: add with append/insert, remove with remove/pop/del.
A for loop can process every item in a list.
The in keyword checks whether a value exists in a list.

Did You Know?

Fun Facts About Lists

A few fun tidbits to impress your friends.

Lists Can Mix Types

A single Python list can hold numbers, strings, and booleans all at once — no type restriction required.

Foundation of Data

Lists are the building block for many other data tools in Python, including data science libraries like pandas.

Dynamic Sizing

Unlike arrays in some languages, Python lists automatically grow or shrink as you add or remove items.

List Comprehensions

Advanced Python lets you build a list in a single line, like [x*2 for x in nums] — a topic for later!
